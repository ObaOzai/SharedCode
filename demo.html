Uniform Probability on Rectangle [0,1] × [0,2]

Understanding the Problem

Sample Space: The rectangular region [0,1] × [0,2]

Total Area: 1 × 2 = 2

Probability Law: For any event E, P(E) = Area(E) / 2

Part (a): The event x = y

The event "x = y" represents all points where both coordinates are equal.

This forms a line from (0,0) to (1,1) within our rectangle.

Since a line has zero area in 2D space:

P(x = y) = 0 / 2 = 0

Part (b): The event x ≥ y

The event "x ≥ y" represents all points where the x-coordinate is greater than or equal to the y-coordinate.

This forms a triangle with vertices at (0,0), (1,0), and (1,1).

Area calculation:

Area = ½ × base × height = ½ × 1 × 1 = ½
P(x ≥ y) = ½ / 2 = ¼ = 0.25

Part (c): The event x² ≥ y

The event "x² ≥ y" represents all points below or on the parabola y = x².

We need to find the area under the curve y = x² from x = 0 to x = 1.

Using calculus:

Area = ∫₀¹ x² dx = [x³/3]₀¹ = 1/3 - 0 = 1/3
P(x² ≥ y) = (1/3) / 2 = 1/6 ≈ 0.16667
